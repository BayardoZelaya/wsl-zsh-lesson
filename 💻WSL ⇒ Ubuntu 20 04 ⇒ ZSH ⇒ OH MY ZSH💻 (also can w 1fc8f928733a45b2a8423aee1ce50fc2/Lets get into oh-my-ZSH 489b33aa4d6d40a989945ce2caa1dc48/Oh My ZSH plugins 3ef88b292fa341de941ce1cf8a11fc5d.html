Oh My ZSH plugins

Here we will go over how to install plugins...

Click here to see more ZSH plugins there are so many to go over so I will show 3 of the most used.

  • first we will install the git plugin to oh my zsh:
nano ~/.zshrc
  • this opens up text editor in ~/.zshrc and you should see the following:
  • then use your down-arrow key to scroll to the line that says plugins=( ):
  • inside plugins type: 
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
  • this sets the plugins in your zsh profile
    • press ctrl + x or command + x to exit, type 'y' to save and press 'enter' or 'return' keys
  • then git clone the zsh-autosuggestions and zsh-syntax-highlighting repos with:
git clone https://github.com/zsh-users/zsh-autosuggestions.git $ZSH_CUSTOM/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git $ZSH_CUSTOM/plugins/zsh-syntax-highlighting
  • finally do:
source ~/.zshrc
  • and if you type a known command into zsh you should see new syntax highlighting in your terminal and autosuggestions plug in working.
